## QUICK STARTING GUIDE:
1. Open the program directory through Terminal, and navigate to src through the following command:

	```sh
    cd src
    ```
2. Compile using the following command:

    ```sh
    javac Main.java
    ```

3. Run the program, using the following command.

	```sh
	java Main
	```

4. Enjoy.
